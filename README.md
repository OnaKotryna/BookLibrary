# BookLibrary

Console app to manage books

## Available commands
  - Add a new book
  - Take a book 
    - max 2 months
    - no more than 3
  - Return a book
  - View a list of books with preferred filter 
    - by author
    - by category
    - by language
    - by ISBN
    - by name
    - by availability (taken or not)
  - Remove a book
